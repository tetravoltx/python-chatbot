A minimalist, terminal-inspired desktop chatbot application built with Python and Tkinter, powered by Google's Gemini AI.

Features:
  1. Terminal-Inspired Interface - Clean, minimalist black & white design with Linux command-line aesthetics
  2. Google Gemini Integration - Powered by Gemini 2.5 Flash for intelligent responses
  3. Keyboard-Driven - Press Enter to send, Shift+Enter for new lines
  4. Monospace Typography - JetBrains Mono/Consolas fonts for that authentic terminal feel
  5. Session Management - Start new chat sessions with a single click
  6. Error Handling - Graceful error messages for API issues
  7. Lightweight - No heavy dependencies, runs on pure Python + Tkinter
